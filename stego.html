arg::stego_kit

Hide clues where players won't think to look — inside image pixels, inside invisible characters in ordinary text. Every tool has a reveal mode so you can confirm the secret round-trips before you drop it.

image steganography (LSB)

Embed a secret message in the least-significant bits of a PNG's pixels. The image looks identical; the message rides along invisibly. Export re-encodes as PNG (lossless) — never JPEG, which would destroy the bits.

drop an image here, or click to choose

zero-width unicode

Encode a hidden message into invisible characters, then splice it into ordinary visible text. Paste the result anywhere — a profile blurb, a forum post, a Discord message — and it looks completely normal. Copy/paste preserves the secret.

trailing whitespace

Encode the message as spaces (bit 0) and tabs (bit 1) appended to the end of a line. Looks like an empty line; survives most plain-text copy/paste. Great for a clue hidden in a README, a code comment, or a pastebin drop.

The whitespace is invisible on screen — download the .txt to keep it intact.
